Binair stelsel

Wat is het binair stelsel?

Normaal gebruiken we een tientallig getallenstelsel, maar een binair getallenstelsel wordt ook gebruikt, vooral computers werken ermee, daarom gebruiken we het in de informatica. Het binair stelsel is een stelsel dat bestaat uit een reeks van de cijfers 0 en 1. Het is dus een tweegetallenstelsel. Alle getallen schrijf je dus met een combinatie van die 2 getallen. Door de 0 en 1 in een bepaalde volgorde te zetten krijg je getallen.

Bits en bytes

De 1 en 0 die we in het binair stelsel gebruiken noemen we bits. Er worden vaak 8 bits gebruikt in een reeks (daarom is het hoogste getal 255). Die reeks (van de getallen 0 en 1) is een byte. De getallen zijn opgebouwd uit machten van 2.

Positie van een bit

De waarde van een bit wordt bepaald door zijn positie. Er wordt van rechts naar links gekeken, dus de bit die helemaal aan de rechterkant staat voor 2⁰, die links ervan 2¹, die links daarvan 2², enzovoort.

Van binair naar decimaal rekenen

Zo zet je een binair getal om naar een decimaal getal, met als voorbeeld het binaire getal 11010. Eerst begin je met de waarde berekenen van de meest rechtse bit (positie 0), dit is 0 x 2⁰ = 0. Daarna de waarde van die links ervan (positie 1), dit is 1 x 2¹ = 2. Daarna de waarde van die links daarvan (positie 2), dus 0 x 2² = 0. Dan die links ernaast (positie 3), dus 1 x 2³ = 8. Dan als laatst degene helemaal aan de linkerkant (positie 4), dit is 1 x 2⁴ = 16. Nu tel je de waarde van elk bit op, dit geeft 0 + 2 + 0 + 8 + 16 = 26. Dus het decimale getal is 26.

Van decimaal naar binair rekenen

Zo zet je een decimaal getal om naar een binair getal, met als voorbeeld het decimale getal 542. Je deelt het getal steeds door 0 en noteert de rest ervan. De rest noteer je dan weer van rechts naar links. 542:2 = 271, rest 0. 271 : 2 = 135 rest 1. 135:2= 67 rest 1. 67:2= 33 rest 1. 33:2= 16 rest 1. 16:2 = 8 rest 0. 8:2 = 4 rest 0. 4:2 = 2 rest 0. 2:2 = 1 rest 0. 1:2 = 0 rest 1. De rest opschrijven van rechts naar links geeft het binaire getal 100011110.

ASCII

Met behulp van ASCII krijgt elk teken een binaire code, waardoor computers informatie kunnen opslaan en manipuleren. In de ASCII tabel kun je zien hoe elk teken genoteerd wordt.
